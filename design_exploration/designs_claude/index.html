Design Exploration - Claude Variants

Ten distinct design systems exploring workout flow interfaces. Each variant implements the exact same UX flow contract with radically different aesthetic directions. Designs 06-10 prioritize dark themes, readability, and low cognitive load.

UX Flow Contract

Main Page -> Start Workout -> Select Exercise -> Progress Page -> Enter Result Data -> Timer Starts -> Input Reps/Weight -> Done -> Rest Countdown -> Auto-return -> Loop until complete
DESIGN 01

Brutalist Terminal

Raw command-line aesthetic inspired by retro computing. Green phosphor CRT glow, monospace typography, and ASCII-influenced borders. Scanline overlay and no rounded corners anywhere.

Monospace CRT Effect Green Phosphor ASCII
DESIGN 02

Luxury Editorial

High-fashion magazine aesthetic meets fitness tracking. Refined serif typography, dramatic black and cream contrast with gold accents. Every interaction feels considered and premium.

Serif Gold Accents Editorial Premium
DESIGN 03

Industrial Blueprint

Technical drawing aesthetic inspired by engineering schematics. Grid paper background, corner brackets, measurement annotations. Precision meets interface design with cyan and orange accents.

Grid Paper Technical Cyan/Orange Schematic
DESIGN 04

Analog Cassette

Retro audio equipment aesthetic. VU meter progress bars, warm browns and ambers, tactile button shadows. Physical depth through inset shadows and raised elements with subtle noise texture.

VU Meters Warm Tones Tactile Audio
DESIGN 05

Swiss Modernist

Helvetica-era inspired design. Bold primary colors (red, blue, yellow), structured grids, asymmetric layouts. Maximum information clarity through systematic visual hierarchy. No decoration.

Primary Colors Bold Borders Condensed Type Grid

BATCH 2: DARK + READABLE

DESIGN 06

Tactical HUD

Military sim inspired clean data readouts. Dark charcoal background with teal accent for high contrast. Rajdhani/Oxanium fonts for precision without harshness. Function over flash.

Dark Mode High Contrast Teal Accent Clean
DESIGN 07

Neon Arcade

Classic arcade cabinet aesthetic. Deep blacks with pink/cyan neon glow. Pixel font for headers, clean sans for data. Focused glow effects that don't overwhelm. Readability first.

Dark Mode Neon Glow Pink/Cyan Arcade
DESIGN 08

Mission Control

NASA/SpaceX inspired mission control aesthetic. Professional dark interface with blue accent hierarchy. Space Grotesk + Space Mono for precision data. Clean, organized, trustworthy.

Dark Mode Blue Accent Monospace Data Professional
DESIGN 09

Stealth Protocol

Night vision / tactical ops aesthetic. Green phosphor on pure black, military precision. Splinter Cell meets gym. Status-driven feedback, minimal decoration, maximum clarity.

Dark Mode Green Phosphor Tactical Minimal
DESIGN 10

Racing Telemetry

F1/racing game inspired dashboard. Bold Orbitron display font, red/orange energy accents. Precision timing displays, dashboard-style data readouts. Fast and focused.

Dark Mode Red/Orange Bold Type Dashboard

Acceptance Checklist

  • UX flow exactly matches contract above
  • Timer starts at Enter Result Data click
  • Rest page auto-returns on timer completion
  • Exercise lock-in until all sets complete
  • 10 distinct design systems (5 original + 5 dark/readable)
  • Designs 06-10: dark mode only, readability-first
  • All files open directly in browser without build tools
  • Mobile + desktop responsive
  • Keyboard/focus accessibility
  • Rest timer set to 3 seconds for testing
